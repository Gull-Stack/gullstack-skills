# Three Kings Sales Talk (pointer)

The canonical skill lives in its own repo: `Gull-Stack/ThreeKingsSalesTalkMethod`.

1. If the repo is already available locally (e.g. `/workspace/threekingssalestalkmethod` or `~/.claude/skills/three-kings-salestalk`), read its `SKILL.md` and follow it — it directs you through `FRAMEWORKS/`, `OBJECTION-MATRIX.md`, `EMAIL-RULES.md`, and the relevant `APPLICATIONS/` file.
2. If not, clone it first: `git clone https://github.com/Gull-Stack/ThreeKingsSalesTalkMethod` (in a Claude remote session, add the repo with the add_repo tool instead).

Core protocol if the repo is temporarily unreachable: every buyer-facing message must pass Voss (accusation audit, mirrors, labels), Belfort (every sentence moves toward close or a Three-Tens trust), and Miller (customer is the hero, you are the guide, name the villain) simultaneously; after every fact add the WizGat emotional outcome or cut the fact. Hard rules: never "I", no questions in cold outreach, short paragraphs, directive close.
